Mask with Unreadable Texture :: Error :: Raycast Threshold greater than 0 can't be used on Soft Mask with texture 'Rotate' because it's not readable. You can make the texture readable in the Texture Import Settings.
Mask with non-2D Texture :: Error :: Raycast Threshold greater than 0 can't be used on Soft Mask with texture 'RenderTexture' because it's not a Texture2D. Raycast Threshold may be used only with regular 2D textures.
